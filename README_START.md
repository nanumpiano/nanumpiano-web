# 범용 프로젝트 문서 묶음 사용법

1. 이 폴더를 새 프로젝트 루트에 푼다.
2. 프로젝트명을 `XXX 프로젝트`에서 실제 이름으로 바꾼다.
3. 정본 6개 문서를 먼저 채운다.
4. 그다음 도구별 세팅 파일을 맞는 위치에 둔다.
5. 마지막으로 아래처럼 첫 지시를 준다.

예시:
- 우리는 XXX 프로젝트를 진행한다.
- 먼저 CONSTITUTION.md, PRINCIPLES.md, STRUCTURE.md, AI.md, SOURCES.md, WORK_LOG.md를 읽고 현재 상태를 확인하라.
- 그 다음 원리 이해와 실행 계획부터 제안하라.
- 바로 구현하지 말고 승인 후 진행하라.
