Antigravity의 잠재력을 200% 활용할 수 있는 고급 기능, 설정 팁, 그리고 유용한 MCP(Model Context Protocol) 서버들을 정리해 드립니다. 쿼터(Quota) 문제를 관리하면서도 작업 효율을 극대화할 수 있는 실전 노하우입니다.

1. 효율적인 쿼터(Quota) 관리 전략
사용자께서 가장 고민하시는 쿼터 부족 문제를 완화하는 방법입니다.

모델 이원화 (Multi-Agent Orchestration):
Gemini 3 Flash: 단순 코드 수정, 파일 구조 변경, 웹 리서치 등 가벼운 작업에 사용 (쿼터 소모 적음).
Gemini 3 Pro / Claude 3.5 Sonnet: 복잡한 로직 설계, 버그 디버깅, 전체 아키텍처 수립 등 핵심 작업에만 집중 투입.
MCP 활용으로 컨텍스트 최적화: AI가 직접 데이터베이스나 외부 문서를 읽게 하여, 긴 텍스트를 대화창에 복사해 넣는 과정에서 발생하는 토큰 낭비를 줄이세요.
2. 안티그래비티의 숨겨진 유용한 기능
① 에이전트 매니저 (Manager View) & 병렬 작업
채팅창 상단의 에이전트 추가 버튼을 눌러보세요. 한 명은 '데이터 분석'을 시키고, 다른 한 명은 그동안 'UI 디자인'을 시키는 등 업무를 분담(병렬 처리)할 수 있습니다.
② 자율 브라우저 테스트 (Autonomous Browser)
에이전트에게 **"내가 짠 코드를 로컬 서버에서 실행하고 브라우저로 직접 테스트해서 오류를 잡아줘"**라고 시키세요. AI가 직접 브라우저를 띄워 클릭하고 스크롤하며 레이아웃이나 기능 오류를 스스로 치유(Self-healing)합니다.
③ .antigravityignore 파일 활용
.gitignore와 유사하게, 에이전트가 읽지 않아도 되는 대용량 라이브러리나 로그 폴더를 제외하세요. 인덱싱 속도가 빨라지고 AI의 판단력이 정확해집니다.
3. 추천 MCP(Model Context Protocol) 서버
안티그래비티를 외부 도구와 연결하는 '확장 플러그인' 같은 개념입니다.

GitHub MCP: 이슈 생성, PR 검토 등을 에이전트가 직접 수행하게 합니다.
Sequential Thinking MCP: 복잡한 문제를 해결할 때 에이전트가 '단계별 사고 과정'을 거치도록 강제하여 정답률을 높입니다.
Fetch MCP: 최신 기술 문서나 API 가이드를 실시간으로 웹에서 긁어와 분석하게 합니다.
SQL/Database MCP: DB에 직접 쿼리를 날려 데이터를 분석하고 보고서를 쓰게 합니다.
4. 마스터 가이드 업데이트 (실전 스킬 추가)
방금 찾은 내용 중 가장 유용한 **'에이전트 위임 및 MCP 활용법'**을 기존 가이드에 보강해 드립니다.
